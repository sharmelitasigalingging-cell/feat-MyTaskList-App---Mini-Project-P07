 DESKP:**MyTaskList App** adalah aplikasi **task manager / to-do list** harian berbasis React Native.

Simpelnya: KITA bisa **catat semua tugas/kegiatan**, terus kelola mereka — tandai yang udah selesai, hapus yang udah beres, atau filter mana yang masih aktif.
**Fitur utamanya:**

- Tambah task baru dengan teks + pilih prioritas (Tinggi/Sedang/Rendah)
- Centang task kalau udah selesai
- Hapus task satu-satu atau sekaligus
- Filter task: Semua / Aktif / Selesai
- Progress bar yang nunjukin berapa task udah kelar dari total
- Validasi input biar ga bisa submit yang kosong

**Analoginya** kayak sticky notes digital di HP KAMU— tulis apa yang mau KAMU lakuin hari ini, centang kalau udah beres, hapus kalau udah ga perlu.
